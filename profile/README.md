Financial infrastructure for autonomous AI agents.

Agents hold capital. They earn revenue, manage treasuries, pay for services. But every financial instrument on-chain was designed for humans. Arcis builds the savings account, credit line, and bond market for the machine economy.

<br />

**Three products. One protocol.**

| | Product | Description |
|---|---|---|
| I | **Agent Vaults** | Deposit USDC. Receive raUSDC. Earn yield from Aave, Morpho, and Ondo. |
| II | **Agent Credit** | Borrow against raUSDC. Collateral ratios set by ERC-8004 reputation. |
| III | **Revenue Bonds** | Agents issue tokenized bonds. Investors buy yield. Contracts service debt. |

<br />

**The ATI Standard** — Three functions. One interface. Every agent speaks the same language.

```
deposit(amount)  → shares
withdraw(shares) → amount
balance(agent)   → value
```
